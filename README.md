# SIAM X ICP | Look Who Committed

SIAM X ICP | Look Who Committed is a beginner friendly project created to help students learn Git, GitHub, and open source contribution by actually doing it.

Instead of only watching tutorials or reading guides, contributors learn the real workflow by forking a repository, adding their details, committing changes, and opening a pull request.

---

## Why this project exists

Open source can feel intimidating when you are starting out.
This project is meant to make the first step simple and welcoming.

By contributing here, you will learn:
- The basics of Git and GitHub
- How forks and pull requests work
- How open source collaboration happens
- How to make your first contribution

---

## How to get your contributor card

Follow these steps carefully.

---

### Step 1: Fork the repository

Click the Fork button on the top right of this repository.
This will create your own copy of the project on GitHub.

---

### Step 2: Clone your fork

Clone the repository to your local machine.

git clone https://github.com/your-username/look-who-committed.git  
cd look-who-committed

---

### Step 3: Add your file

Inside the users directory, create a new JSON file.

File name format:
your_name_rollno.json

Example:
rahul_cs_102.json

---

### Step 4: JSON format

Use the following format in your file.
```
{
  "name": "Your Name",
  "rollno": "CS_[roll number]",
  "pokemon": "pikachu",
  "github": "https://github.com/your_github_username",
  "linkedin": "https://www.linkedin.com/in/your_linkedin_username",
  "twitter": "https://twitter.com/your_twitter_username",
  "bio": "Your short bio goes here."
}
```

Make sure the JSON is valid and the file name follows the format.

---

### Step 5: Commit your changes

Add and commit your file.

git add users/your_name_rollno.json  
git commit -m "Add contributor: Your Name"

---

### Step 6: Push your changes

Push the commit to your fork.

git push origin main

---

### Step 7: Create a pull request

Go to your fork on GitHub.
Click Compare and Pull Request.
Add a short description and submit it.

---

## What happens next

Once your pull request is reviewed and merged:
- Your contributor card will appear on the site
- You become part of the SIAM X ICP open source showcase
